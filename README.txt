# Book Database OOP simulation

This is a beginner Java project for managing a book database using external Gson library.

## Technologies used
- Java
- Gson.jar


## Features
- Add new books
- Update existing books
- Delete books
- View book list
- Search book by writer's name
- Search book by genre
- Saving a book into file or json
- Loading a book from file or json

## How to run
1. Clone the repository
2. Open a project in Intellij Idea
3. Run the application
